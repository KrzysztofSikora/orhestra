Gallery Pro is an advanced plugin with a list of Features and options for adding and editing multiple images for different views. It is All Device Compatible(Responsive).

== Description ==

With Gallery Pro you can Add unlimited images on your blog site. Design various types of image, video, link gallery using this plugin. Gallery Pro provide you light box image preview and two gallery layout with various fonts. 

Using **[WG id=123]** shortcode, you can publish image or photo gallery into any Page or Post in your WordPress sites. You can use multiple gallery shortcode on multiple pages and posts to display various images galleries.

== Installation ==

1. Upload the entire `gallery-pro` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Widgets in Appearance Menu and activate widget.
