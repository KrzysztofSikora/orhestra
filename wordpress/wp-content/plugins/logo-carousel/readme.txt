== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use logo carousel in your post, page, widget or anywhere you like. How to create logo carousel? lets see.


1. Upload the folder logo-carouselÂ to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
4. Add logo going to wp-admin -> Logo Carousel -> Add New Logo.
5. Call shortcode to your WordPress editor directly, by using 
`
[logo-carousel id="01"]
`
Call shortcode to your php template file using 
`
<?php  echo do_shortcode('[logo-carousel id="01"]'); ?>
`

Now save your page, then visit the newly created page & see something awesome. :)
